## Controls

`W` -> move forward  
`S` -> move backward  
`A` -> move left  
`D` -> move right  
`UP` -> move up  
`DOWN` -> move down  
`RIGHT` -> agent moves right  
`LEFT` -> agent moves left   
`ESC` -> close the program   
`SCROLL` -> zoom in/out
